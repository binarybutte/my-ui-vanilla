초기화 버튼

input[type='search'] 일때 각각의 브라우저에서는 초기화 버튼을 자동 생성한다. 버튼 디자인의 통일을 위해 input 옆에 .btn-input-clear 버튼을 생성해주고, css에서 input[type='search'] 의 초기화 버튼 속성을 없앤다. (하기 예시를 위해 주석처리함)
/* input[type="search"]::-webkit-search-cancel-button, input[type="search"]::-webkit-search-decoration{-webkit-appearance:none;} */
