Inputs & Textarea

기본

취소(초기화) 버튼

input[type='search'] 일때 각각의 브라우저에서는 초기화 버튼을 자동 생성한다.
css에서 input[type='search'] 의 초기화 버튼 속성을 없애고, 버튼 디자인의 통일을 위해 input 옆에 .input-clear-btn 버튼을 생성하여 만든다
input[type="search"]::-webkit-search-cancel-button, input[type="search"]::-webkit-search-decoration{-webkit-appearance:none;}

Datalist

브라우저 지원 확인 필요. 셀렉트박스와 비슷하나 목록 표시에 디자인 차이가 있다.

Textarea

oninput 이벤트 적용: 키보드 입력뿐만 아니라 마우스로 붙여넣기, 드래그 앤 드롭, 자동 완성, 음성 입력 등 모든 입력 방식을 감지함.
텍스트가 변경될 때마다 실행되므로 가장 정확하게 카운트를 반영할 수 있음.
(keydown, keyup보다 좀 더 직관적이고 정확하다. keydown, keyup 등은 특정 키 이벤트 감지할 때 사용 권장)
