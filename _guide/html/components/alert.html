Toast

주로 화면의 상단이나 하단에 잠시 나타나는 단순한 알림 메시지. 자동으로 사라지며, 사용자가 이를 닫을 필요는 없다.
사용자가 특정 작업을 완료했을 때 간단한 정보를 표시 (ex: 저장되었습니다, 업데이트 완료)

Snackbar

Toast와 비슷하지만, 사용자와의 상호작용을 더 중요시. 닫기 버튼등이 있을 수 있다.
사용자가 버튼을 클릭하거나 특정 액션을 취할 수 있는 경우가 많다. (ex: 실행취소 버튼)
