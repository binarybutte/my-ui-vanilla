Selector

특정 문자열로 시작 [class^='input-'] { ... }
특정 문자열을 포함 [class*='input'] { ... }
특정 문자열로 끝 [class$='-input'] { ... }

JS 활용

JavaScript에서 dd로 시작하는 클래스명을 가진 요소들의 클래스를 제거

<div class="dd-item">Item 1</div>
<div class="dd-menu dd-active">Item 2</div>
<div class="normal">Item 3</div>

<script>
	// 방법 1
	document.querySelectorAll("[class^='dd']").forEach(element => {
		element.classList.forEach(className => {
			if (className.startsWith("dd")) {
				element.classList.remove(className);
			}
		});
	});

	// 방법 2: 모든 요소를 검사하고, classList를 순회하면서 dd로 시작하는 클래스 삭제
	document.querySelectorAll("*").forEach(element => {
		element.classList.forEach(className => {
			if (className.startsWith("dd")) {
				element.classList.remove(className);
			}
		});
	});
</script>

Animation

duration(1) 과 delay(2) 순서는 맞춰야 한다.

  • https://developer.mozilla.org/ko/docs/Web/CSS/animation
animation: name | duration | easing-function | delay | iteration-count | direction | fill-mode | play-state

ex: aniName animation: 3s ease-in 1s 2 reverse both paused;
ex:  animation: 3s linear 1s slidein;
// 애니메이션 두 개
ex:  animation: 3s linear slidein, 3s ease-out 5s slideout; 

@keyframes aniName {  
	from { .. }    
	to { .. }  
}
  • name: @keyframes 애니메이션 이름 정의
  • duration: 시작 ~ 완료되기까지의 소요 시간
  • timing-function: 효과의 속도 패턴 (변환 속도) ease, ease-in, ease-out, ease-in-out, linear
  • delay: 시작 시점의 지연시간을 지정
  • iteration-count: 재생 횟수 기본값 1, 무한반복 infinite
  • direction: 진행하는 방향

    normal: 순방향 재생
    alternate: 순방향으로 시작하여 역방향과 번갈아 재생
    reverse: 역방향 (to ~ from) 으로 재생
    alternate-reverse: 역방향으로 시작하여 순방향과 번갈아 재생

  • fill-mode: 종료된 후 상태

    none: 기본값. 애니메이션 실행 전 상태로 되돌리고 종료
    forwards: 마지막 키프레임을 유지
    backwards: 애니메이션이 재생되기 전에 미리 적용
    both: forwards backwards 둘 다 적용

  • play-state: 재생 여부 (재생 또는 중지) 를 지정

    running: 재생
    paused: 정지
